微信快速开发框架（WXPP QuickFramework）已更新至V4.0版本上线

您可以用来做什么？
您可以使用此框架快速开发微信公众平台，目前V3.0已经支持高级接口的开发。

使用教程可以参考我的博客：http://inday.cnblogs.com

如果在使用中有任何问题，可以Email给我：james#taogame.com
